## HTTP Request Data

### GET - Query Parameter

- 메시지 바디 없이, URL의 쿼리 파라미터에 데이터를 포함해서 전달
- 예) 검색, 필터, 페이징 등에서 많이 사용

### POST - HTML Form

- 메시지 바디에 쿼리 파라미터 형식으로 전달
- 예) 회원가입, 상품 주문, HTML Form 사용

### HTTP message body에 데이터를 직접 담아서 요청

- HTTP API에서 주로 사용, JSON, XML, TEXT
- 데이터 형식은 주로 JSON 사용
- POST, PUT, PATCH
