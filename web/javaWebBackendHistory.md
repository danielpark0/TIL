## 현재 사용 기술

### 애노테이션 기반의 스프링 MVC 등장

- @Controller
- MVC 프레임 워크의 춘추 전국 시대 마무리

### 스프링 부트의 등장

- 스프링 부트는 서버를 내장
- 과거에 서버에 WAS를 직접 설치하고, 소스는 War 파일을 만들어서 설치한 WAS에 배포
- 스프링 부트는 빌드 결과(Jar)에 WAS 서버 포함 -> 빌드 배포 단순화
